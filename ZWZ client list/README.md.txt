# Customer Analysis Dashboard (Free + Real-Time)

Analyze your customer orders directly from Google Sheets in a web dashboard.

## ✅ Setup

1. Create a Google Sheet with your order data.
2. Share it → “Anyone with the link can view.”
3. Copy your Sheet ID from the URL.
4. Paste it into the dashboard input box.

## 🧠 Features

- Real-time data from Google Sheets  
- View customer spending, frequency, and totals  
- Sort and analyze by phone (your unique key)  
- 100% Free (GitHub Pages + Google Sheets)

## 🌍 Deployment (Free Hosting)

1. Create a free GitHub account at [github.com](https://github.com)
2. Create a new repo (name it anything)
3. Upload `index.html`, `app.js`, and `README.md`
4. Go to **Settings → Pages**
5. Under “Source,” choose **Deploy from branch → main**
6. Click **Save**

Your site will be live at:  
`https://your-username.github.io/your-repo-name`
